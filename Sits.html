Project Overview

Contributed to the development and launch of a corporate website for Sampath IT Solutions, a subsidiary of Sampath Bank PLC, Sri Lanka. The website was built using WordPress and Elementor Pro to reflect the company's technological expertise and professional branding. It features detailed service pages, client case studies, and a modern UI with responsive design. The goal was to position SITS as a leading IT service provider in Sri Lanka's financial and enterprise sectors.

Key Highlights

  • Corporate Design: Clean and modern design aligned with Sampath Bank’s brand identity.
  • Built on WordPress + Elementor Pro: Enables easy content management and fast development.
  • Mobile-Responsive: Fully optimized for desktop, tablet, and mobile viewing.
  • SEO Optimization: Ensured visibility in search engines with structured metadata and speed optimization.
  • Service-Oriented Pages: Clear navigation with sections for services, solutions, and client success stories.
  • Secure Hosting and SSL: Deployed on a secure server with HTTPS and performance optimizations.
