Project Overview

Data Vision Studio is a full-stack marketing application developed for Sensia Ventures Group. It streamlines campaign management by integrating inventory systems, media planning tools, and advanced analytics under a unified dashboard. Using OpenAI and data science models, the platform predicts marketing outcomes, optimizes strategies, and automates reporting to drive smarter decisions.

Key Highlights

  • Campaign Media Planning: Enables marketers to create, compare, and manage media campaigns across digital and offline channels.
  • Inventory Integration: Real-time stock tracking integrated into campaign tools for seamless alignment with product availability.
  • AI-Powered Predictions: Utilizes OpenAI and machine learning to forecast performance for email and direct mail campaigns.
  • Analytics Dashboard: Interactive charts, KPIs, and insights to track ROI, user behavior, and engagement trends.
  • Collaborative Tools: Built-in team workflows for campaign approval, budgeting, and reporting across departments.
  • Responsive UI: Built with React for a fluid interface across mobile, tablet, and desktop devices.
  • Secure Architecture: Django-based backend with API authentication, role-based access, and AWS deployment.
  • SEO & Performance Optimization: Structured metadata, fast load speeds, and Google Search Console integration.
